# Probability Notes

## Conditional Probability

Formula:

P(A|B) = P(A ∩ B) / P(B)

Use when:
- Event B is given.

---

## Law of Total Probability

Formula:

P(B) = Σ P(B|Ai)P(Ai)

Use when:
- P(B) is not directly given.
- The sample space can be partitioned into cases.

---

## Bayes' Theorem

Formula:

P(A|B) = P(B|A)P(A) / P(B)

Workflow:
1. Find P(B) using Total Probability.
2. Apply Bayes' Theorem.

---

## Independence

Key Property:

P(A ∩ B) = P(A)P(B)

Equivalent:

P(A|B) = P(A)

P(B|A) = P(B)

Meaning:
- Knowing one event does not affect the other.

---

## Dependence

Formula:

P(A ∩ B) = P(A)P(B|A)

P(A|B) = P(A ∩ B) / P(B)

Meaning:
- One event affects the probability of the other.

---

# Probability Models

## Sample Space

Ω = set of all possible outcomes.

---

## Union Rule

Two Events:

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

Three Events:

P(A ∪ B ∪ C)

= singles

− pairwise intersections

+ triple intersection

# Counting Techniques

## Multiplication Principle

Formula:

n₁ × n₂ × ... × nₖ

Use when:
- A process has multiple stages.
- Choices are made sequentially.

---

## Permutation

Formula:

nPr = n! / (n − r)!

Special Case:

nPn = n!

Use when:
- Order matters.

Examples:
- Rankings
- Arrangements
- Seating

---

## Combination

Formula:

nCr = n! / [r!(n − r)!]

Use when:
- Order does not matter.

Examples:
- Committees
- Teams
- Selections

---

## Multinomial Coefficient

Formula:

n! / (n₁!n₂!...nₖ!)

Constraint:

n₁ + n₂ + ... + nₖ = n

Use when:
- Objects repeat.

Example:
- TORONTO
- MISSISSIPPI

---

## Hypergeometric Distribution

Formula:

C(n₁,r₁) · C(n₂,r₂)
-------------------
      C(N,r)

Where:

N = n₁ + n₂

r = r₁ + r₂

Use when:
- Sampling without replacement.

---

# Quick Recognition Guide

| Question Type | Method |
|--------------|---------|
| Given B | Conditional Probability |
| Need P(B) | Total Probability |
| Reverse Conditional | Bayes |
| Independent Events | P(A∩B)=P(A)P(B) |
| Order Matters | Permutation |
| Order Doesn't Matter | Combination |
| Repeated Objects | Multinomial |
| Without Replacement | Hypergeometric |

---

# Reminders

- Bayes often requires Total Probability first.
- Independence ⇒ conditioning changes nothing.
- Always ask: "Does order matter?"
- 0! = 1
